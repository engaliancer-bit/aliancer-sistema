# Portal do Cliente - Guia Completo

## Visão Geral

O Portal do Cliente permite que seus clientes do escritório de engenharia e topografia acessem informações sobre seus imóveis, projetos, documentos e serviços diretamente pelo celular ou computador, como se fosse um aplicativo.

## Funcionalidades para o Cliente

### O que o cliente pode fazer no Portal:

1. **Acompanhar Imóveis**
   - Ver todos os imóveis cadastrados
   - Acessar informações detalhadas de cada imóvel
   - Visualizar documentos anexados
   - Verificar datas de vencimento

2. **Acompanhar Projetos**
   - Ver status de projetos em andamento
   - Acompanhar percentual de progresso
   - Ver fase atual do projeto
   - Verificar previsão de conclusão

3. **Aprovar Serviços**
   - Receber orçamentos para aprovação
   - Aprovar ou rejeitar serviços
   - Ver detalhes de valor e prazo
   - Histórico de aprovações

4. **Solicitar Serviços**
   - Fazer novas solicitações
   - Definir urgência
   - Anexar informações relevantes
   - Acompanhar status da solicitação

5. **Receber Notificações**
   - Alertas de vencimento de documentos
   - Atualizações de projetos
   - Novos orçamentos para aprovação
   - Avisos importantes

## Como Usar - Passo a Passo

### Passo 1: Acessar o Portal do Cliente

1. No sistema, acesse: **Escritório de Engenharia e Topografia**
2. Clique em: **Portal do Cliente** (ícone de smartphone)

### Passo 2: Gerar Token de Acesso para o Cliente

1. Na lista de clientes, localize o cliente desejado
2. Certifique-se de que o cliente tem um **telefone cadastrado**
3. Clique em **"Gerar Token"**
4. Um token único será criado automaticamente

### Passo 3: Enviar o Acesso para o Cliente

Você tem duas opções:

#### Opção 1: Enviar por WhatsApp (Recomendado)
1. Após gerar o token, clique no botão **"WhatsApp"**
2. Uma mensagem automática será criada com:
   - Boas-vindas
   - Instruções de acesso
   - Link direto para o portal
   - Informações sobre validade
3. Revise a mensagem e envie pelo WhatsApp

#### Opção 2: Copiar e Enviar Manualmente
1. Clique no botão de **copiar** ao lado do link
2. Envie o link para o cliente por e-mail ou WhatsApp
3. O cliente pode colar o link no navegador do celular

### Passo 4: Cliente Acessa o Portal

1. Cliente clica no link recebido
2. Sistema valida o token automaticamente
3. Cliente é direcionado para seu dashboard personalizado

### Passo 5: Cliente Instala como App (Opcional)

No celular do cliente:
1. Abrir o link no navegador (Chrome/Safari)
2. Quando aparecer o aviso de instalação, clicar em "Instalar"
3. O ícone do app aparecerá na tela inicial
4. Cliente pode abrir como aplicativo nativo

## Gerenciando Aprovações e Notificações

### Criar uma Aprovação de Serviço

1. No Portal do Cliente, clique em **"Nova Aprovação"**
2. Preencha:
   - Cliente
   - Título do serviço
   - Descrição detalhada
   - Valor estimado
   - Prazo estimado (dias)
   - Validade da proposta (dias)
3. Clique em **"Criar Aprovação"**
4. Cliente receberá notificação automática

### Enviar Notificação para Cliente

1. Clique em **"Enviar Notificação"**
2. Preencha:
   - Cliente
   - Tipo (vencimento, projeto, documento, etc)
   - Prioridade (baixa, normal, alta, urgente)
   - Título
   - Mensagem
3. Clique em **"Enviar Notificação"**
4. Notificação aparecerá no portal do cliente

## Segurança e Controle

### Gerenciamento de Acesso

- **Habilitar/Desabilitar Acesso**: Clique no botão de status ao lado do cliente
- **Desativar Token**: Se necessário, desative tokens específicos
- **Validade**: Tokens expiram automaticamente em 90 dias
- **Renovação**: Gere um novo token quando necessário

### Recursos de Segurança

- Token único e criptografado para cada cliente
- Acesso restrito apenas aos dados do próprio cliente
- Tokens podem ser desativados a qualquer momento
- Expiração automática configurável
- Registro de último acesso

## Exemplo de Fluxo Completo

### Cenário: Novo Projeto de Topografia

1. **Escritório cria o projeto** no sistema
   - Cadastra cliente e imóvel
   - Cria projeto de topografia
   - Anexa documentos necessários

2. **Gera acesso ao portal**
   - Clica em "Gerar Token" para o cliente
   - Envia link por WhatsApp

3. **Cliente recebe e instala**
   - Clica no link
   - Instala como app no celular
   - Faz primeiro acesso

4. **Escritório envia orçamento**
   - Cria "Nova Aprovação"
   - Valor: R$ 5.000,00
   - Prazo: 15 dias
   - Cliente é notificado

5. **Cliente aprova**
   - Abre o app
   - Vê notificação de novo orçamento
   - Revisa detalhes
   - Clica em "Aprovar"

6. **Escritório executa**
   - Recebe confirmação da aprovação
   - Atualiza progresso do projeto
   - Cliente acompanha em tempo real

7. **Atualizações contínuas**
   - Escritório envia notificações de progresso
   - Cliente acompanha percentual de conclusão
   - Documentos finais são anexados
   - Cliente acessa tudo pelo app

## Dicas e Boas Práticas

### Para o Escritório:

1. **Cadastre telefones**: Sempre cadastre o telefone do cliente antes de gerar o token
2. **Mensagens claras**: Use títulos e descrições objetivas nas notificações
3. **Mantenha atualizado**: Atualize o progresso dos projetos regularmente
4. **Documentos organizados**: Anexe documentos com nomes descritivos
5. **Validade razoável**: Configure prazos realistas para aprovações

### Para orientar o Cliente:

1. **Instalar como app**: Incentive a instalação para melhor experiência
2. **Ativar notificações**: Peça para permitir notificações do navegador
3. **Guardar o link**: Cliente pode salvar o link nos favoritos
4. **Acessar regularmente**: Incentive verificações periódicas

## Resolução de Problemas

### Token não funciona
- Verifique se o token não expirou
- Confirme se o acesso está habilitado para o cliente
- Gere um novo token se necessário

### Cliente não recebe notificações
- Verifique se as preferências de notificação estão ativas
- Confirme se o e-mail/telefone estão corretos
- Teste enviando uma notificação manualmente

### Documentos não aparecem
- Verifique se "Compartilhar Documentos" está ativado no imóvel
- Confirme se o documento está anexado corretamente
- Verifique se o cliente está visualizando o imóvel correto
