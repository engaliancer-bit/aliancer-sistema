## 🎯 Exemplo de Uso Rápido

### Cenário: Levantamento Topográfico

**1. Criar Projeto**
```
Nome: Levantamento Topográfico - Fazenda Santa Maria
Cliente: João da Silva
Imóvel: Fazenda Santa Maria
Data: 18/01/2026
```

**2. Adicionar Serviços**
```
Serviço: Levantamento Topográfico
Valor Praticado: R$ 4.500,00
```

**3. Adicionar Custos**
```
Deslocamento: R$ 300,00
```

**4. Adicionar Marcos**
```
Marco de Concreto: 4 unidades × R$ 85,00 = R$ 340,00
```

**5. Total Automático**
```
TOTAL GERAL: R$ 5.140,00
```

**6. Salvar**
- ✅ Projeto criado
- ✅ 4 marcos descontados do estoque
- ✅ Tudo pronto!

**7. Receber Pagamento**
```
Data: 18/01/2026
Valor: R$ 2.000,00
Forma: PIX
Conta: Banco do Brasil
```
- ✅ Fluxo de caixa atualizado
- ✅ Saldo: R$ 3.140,00

## 💡 Dicas Rápidas

### ✅ Valores Praticados
Você pode dar descontos editando o "Valor Praticado". O sistema sempre usa esse valor nos cálculos.

### ✅ Múltiplos Serviços
Um projeto pode ter quantos serviços você quiser. Adicione todos os necessários.

### ✅ Custos Extras
Não esqueça de registrar taxas, deslocamento, hospedagem, etc. Isso afeta a rentabilidade.

### ✅ Marcos de Concreto
O sistema verifica o estoque automaticamente. Se não tiver disponível, não deixa salvar.

### ✅ Pagamentos Parciais
Pode registrar quantos pagamentos quiser até quitar o projeto.
