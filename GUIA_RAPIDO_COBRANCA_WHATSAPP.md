# Guia Rápido: Cobrança via WhatsApp

## Configuração Inicial (Fazer Uma Vez)

### 1. Configure a Empresa
```
Módulo Fábrica → Configurações da Empresa
```

Preencha:
- ✅ Nome da Empresa
- ✅ Telefone: (65) 99999-9999
- ✅ Chave PIX: (CPF/CNPJ/Email/Telefone)
- ✅ Dados Bancários: Banco - Ag: XXXX / Conta: XXXXX-X

### 2. Cadastre Telefone dos Clientes
```
Módulo Engenharia → Clientes
```

- Edite cada cliente
- Adicione o telefone com DDD
- Salve

---

## Como Enviar Cobrança

### Passo 1: Acesse a Aba "À Cobrar"
```
Módulo Engenharia → Projetos de Engenharia e Topografia → Aba "À Cobrar"
```

### Passo 2: Identifique o Projeto
- Veja a lista de projetos com saldo devedor
- Identifique o cliente e valor a receber

### Passo 3: Clique no Botão WhatsApp (Verde)
- Na coluna "Ações", clique no primeiro botão verde (WhatsApp)
- Um modal abrirá mostrando a mensagem

### Passo 4: Revise a Mensagem
- Confira valores
- Verifique informações de pagamento
- Confirme dados do cliente

### Passo 5: Envie
Escolha uma opção:

**A) Abrir WhatsApp Web:**
- Clique em "Abrir WhatsApp"
- WhatsApp abrirá com mensagem pronta
- Pressione Enter para enviar

**B) Copiar e Colar:**
- Clique em "Copiar Mensagem"
- Abra WhatsApp manualmente
- Cole e envie

---

## Quando o Projeto NÃO Aparece na Aba "À Cobrar"

O projeto só aparece na aba "À Cobrar" quando:
- ✅ Tem saldo devedor (balance > 0)
- ✅ Status está como "finalizado", "entregue" ou "registrado"

**Soluções:**
- Verifique o status do projeto
- Confirme se há saldo a receber
- Atualize a página

## Quando o Botão WhatsApp NÃO Funciona

O botão precisa:
- ✅ Cliente tem telefone cadastrado

**Solução:**
- Cadastre telefone do cliente

---

## Formato da Mensagem

A mensagem inclui automaticamente:

1. **Identificação**
   - Nome da empresa
   - Nome do projeto
   - Nome do imóvel

2. **Valores**
   - Valor negociado
   - Custos adicionais (detalhados)
   - Resumo financeiro
   - Saldo a receber (DESTAQUE)

3. **Formas de Pagamento**
   - Transferência bancária (com dados)
   - PIX (com chave)
   - Dinheiro
   - Cartão de crédito/débito
   - Cheque
   - Falar com a Aliancer

4. **Contato**
   - Telefone da empresa

---

## Dicas Rápidas

✅ **Sempre revise** a mensagem antes de enviar

✅ **Mantenha atualizado** os dados bancários

✅ **Registre pagamentos** no sistema após receber

✅ **Acompanhe** a resposta do cliente

✅ **Personalize** se necessário (copie e edite)

---

## Alertas Comuns

| Alerta | Solução |
|--------|---------|
| "Cliente não possui telefone" | Cadastre telefone do cliente |
| "Não possui saldo devedor" | Confirme recebimentos no sistema |
| "Cliente não encontrado" | Verifique vinculação do projeto |
