# Instalação Rápida - PWA Sistema Aliancer

## 3 Passos Simples para Instalar

### Passo 1: Acesse o Sistema
Abra o navegador e acesse o sistema normalmente.

### Passo 2: Instale o App
Um banner azul aparecerá automaticamente no canto da tela:

```
┌─────────────────────────────────────┐
│ 📱 Instalar App                     │
│                                     │
│ Instale o Sistema Aliancer no seu  │
│ dispositivo para acesso rápido e    │
│ experiência completa de aplicativo! │
│                                     │
│ [Instalar Agora]  [Depois]         │
└─────────────────────────────────────┘
```

Clique em **"Instalar Agora"**

### Passo 3: Use o App
O ícone aparecerá na sua tela inicial. Clique para abrir!

## Compartilhar Módulos - Tutorial de 30 Segundos

### 1. Acesse "Compartilhar Módulos"
Na tela inicial do sistema, clique no card **"Compartilhar Módulos"** (ícone 🔗)

### 2. Selecione o que Compartilhar
- **Filtre** por categoria: Indústria, Engenharia ou Construtora
- **Clique** nos módulos que deseja compartilhar
- Módulos selecionados ficam **destacados em cores**

**Exemplo: Compartilhar com Vendedor**
```
✅ Clientes
✅ Orçamentos
✅ Vendas
✅ Produtos
```

### 3. Copie o Link ou QR Code
Um link é gerado automaticamente:
```
https://seusistema.com/?shared_modules=customers,quotes,sales,products
```

**Opções:**
- 📋 **Copiar Link** → Cole no WhatsApp
- 📱 **QR Code** → Baixe e compartilhe a imagem
- 🔗 **Compartilhar** → Envie direto por apps

### 4. Envie para a Pessoa
A pessoa que receber:
- Abre o link
- Vê **apenas os módulos selecionados**
- Acesso controlado e personalizado

## Casos de Uso Rápidos

### 1. Vendedor Externo
**Compartilhe:**
- Clientes
- Orçamentos
- Produtos
- Vendas

**Resultado:** Vendedor tem tudo que precisa, sem acesso ao financeiro ou produção.

### 2. Engenheiro de Campo
**Compartilhe:**
- Projetos
- Imóveis
- Acompanhamento

**Resultado:** Acesso focado em projetos de engenharia.

### 3. Administrativo/Financeiro
**Compartilhe:**
- Fluxo de Caixa
- Financeiro
- Relatórios
- Colaboradores

**Resultado:** Visão completa financeira sem acesso à produção.

### 4. Cliente Externo (VIP)
**Compartilhe:**
- Acompanhamento de Obras
- Relatórios (apenas seu projeto)

**Resultado:** Cliente acompanha sua obra em tempo real.

## Perguntas Frequentes (FAQ Rápido)

**Q: Preciso baixar da Play Store?**
A: Não! Instale direto do navegador.

**Q: Funciona em iPhone?**
A: Sim! No Safari, use "Adicionar à Tela de Início".

**Q: Ocupa muito espaço?**
A: Não! Menos de 5 MB geralmente.

**Q: Precisa de internet?**
A: Para usar sim, mas continua funcionando offline após carregar.

**Q: Como atualizar?**
A: Automático! Sem precisar fazer nada.

**Q: Como desinstalar?**
A: Igual qualquer app - pressione e segure, depois "Remover".

**Q: Os links expiram?**
A: Não! Use indefinidamente.

**Q: Posso instalar em vários celulares?**
A: Sim! Sem limite.

## Atalhos de Teclado (Desktop)

Quando o app estiver aberto no desktop:

- `Ctrl + -` - Diminuir zoom
- `Ctrl + 0` - Zoom padrão

## Suporte Rápido

**Problema:** Banner não aparece
**Solução:** Acesse pelo Chrome ou Edge

**Problema:** Offline não funciona
**Solução:** Abra o site pelo menos uma vez com internet

**Problema:** Ícone errado
**Solução:** Desinstale e reinstale

**Problema:** Link não funciona
**Solução:** Copie novamente, pode ter cortado
