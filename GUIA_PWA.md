# Guia do Progressive Web App (PWA) - Sistema Aliancer

## O que foi implementado?

O Sistema Aliancer agora é um **Progressive Web App (PWA)**, permitindo que seja instalado como um aplicativo nativo em qualquer dispositivo!

## Funcionalidades do PWA

### 1. Instalação como App
- **Android**: Ícone na tela inicial, funciona como app nativo
- **iOS**: Adicionar à tela inicial do iPhone/iPad
- **Windows/Mac/Linux**: Instalar via Chrome, Edge, etc.
- **Sem necessidade de lojas de aplicativos** (Google Play, App Store)

### 2. Funcionalidade Offline
- Cache automático de páginas e recursos
- Continua funcionando mesmo sem internet
- Sincronização automática quando voltar online

### 3. Compartilhamento de Módulos
- Selecione módulos específicos para compartilhar
- Gere links personalizados
- Crie QR Codes para acesso rápido
- Compartilhe via WhatsApp, Email, etc.

## Como Instalar o App?

### No Android (Chrome/Edge)

1. Acesse o sistema pelo navegador
2. Um banner aparecerá automaticamente: "Instalar App"
3. Clique em "Instalar Agora"
4. O app será adicionado à sua tela inicial
5. Abra direto pela tela inicial como qualquer outro app

**OU**

1. Abra o menu do navegador (⋮)
2. Selecione "Adicionar à tela inicial" ou "Instalar app"
3. Confirme a instalação

### No iPhone/iPad (Safari)

1. Acesse o sistema pelo Safari
2. Toque no botão de compartilhar (□↑)
3. Role para baixo e selecione "Adicionar à Tela de Início"
4. Dê um nome ao app (sugerido: "Aliancer ERP")
5. Toque em "Adicionar"
6. O ícone aparecerá na tela inicial

### No Windows/Mac/Linux (Chrome/Edge)

1. Acesse o sistema
2. Procure pelo ícone de instalação na barra de endereço (⊕)
3. Clique em "Instalar"
4. O app abrirá em uma janela própria
5. Será adicionado ao menu iniciar/aplicativos

## Como Usar o Compartilhamento de Módulos?

### Passo 1: Acessar a Tela de Compartilhamento
1. Na tela principal, clique em "Compartilhar Módulos"

### Passo 2: Selecionar Módulos
1. Filtre por categoria (Indústria, Engenharia, Construtora)
2. Clique nos módulos que deseja compartilhar
3. Módulos selecionados ficam destacados em cores

### Passo 3: Gerar Link
1. O link é gerado automaticamente
2. Um QR Code também é criado

### Passo 4: Compartilhar
**Opção 1 - Copiar Link:**
- Clique em "Copiar"
- Cole em WhatsApp, Email, etc.

**Opção 2 - QR Code:**
- Clique em "Baixar QR Code"
- Compartilhe a imagem
- Outras pessoas podem escanear com a câmera

**Opção 3 - Compartilhar Direto:**
- Clique em "Compartilhar" (se disponível)
- Escolha o app (WhatsApp, Telegram, etc.)

### Praticidade

- Não ocupa muito espaço
- Funciona em qualquer dispositivo
- Sem necessidade de lojas de apps

### Segurança

- Mesmo nível de segurança da web

## Perguntas Frequentes

**Q: O app ocupa muito espaço?**
A: Não! PWAs ocupam muito menos espaço que apps nativos (geralmente < 5 MB).

**Q: Preciso de internet para usar?**
A: Após a primeira visita, muitas funcionalidades funcionam offline. Porém, para sincronizar dados novos, é necessária conexão.

**Q: Funciona em qualquer celular?**
A: Sim! Android (Chrome/Edge), iPhone (Safari), e todos os sistemas modernos.

**Q: Como desinstalar?**
A:
- **Android**: Pressione e segure o ícone → "Remover" ou "Desinstalar"
- **iOS**: Pressione e segure → "Remover App"
- **Desktop**: Configurações do navegador → Apps Instalados → Desinstalar

**Q: O link compartilhado expira?**
A: Não! Os links de módulos compartilhados não expiram e podem ser usados indefinidamente.

**Q: Posso instalar em vários dispositivos?**
A: Sim! Você pode instalar o app em quantos dispositivos quiser.

**Q: Os dados são sincronizados entre dispositivos?**
A: Sim! Todos os dados são salvos no banco de dados e sincronizados automaticamente.

## Suporte

Para dúvidas ou problemas com o PWA:
- Certifique-se de estar usando um navegador moderno atualizado
- Verifique sua conexão com a internet na primeira instalação
- Em caso de problemas, limpe o cache do navegador e tente novamente
